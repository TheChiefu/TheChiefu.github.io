Expectation 1: Problem Solving

Experience: Figuring out the basics of Game Design

Over the course of the summer I wanted to learn about what profession I want to pursue when I’m older, so I went to find ways to get started on game design. I started out by finding that I need a game engine for all of the game to work and be built in. I knew of a free one named Unity from forever ago (around 5 years ago, in 2010) but it was daunting at the time, so I made it my objective to learn the basics this summer. I started on making a simple level to better understand how to use the game engine in order to make a video game. I need to make the world using 3D models made in a program from Blender (a free 3D Modeling program). I’ve used it before but I learned so much more this summer and am able to make more detailed objects now more than ever.
